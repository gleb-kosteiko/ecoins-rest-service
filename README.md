# ecoins-rest-service

## Requirements:
- Java 8 or higher
- Maven 3.3 or higher

## How to build and to run:
```
mvn clean package spring-boot:run
```
## Local Swagger documentation page:
http://localhost:8080/api/swagger-ui.html
